### Skills to Know About
- `/voice` — Dictate instead of typing. Useful for longer explanations or when you're away from the keyboard.
- `/config` — Open Claude Code settings. Use it to adjust permissions, timeouts, and model preferences.
- `/plugin` — Browse and manage Claude Code plugins. The team uses plugins for extended functionality.
- `/reload-plugins` — Reload plugins after changes without restarting the session.
- `/resume` — Pick up where you left off in a previous session. Handy when you get interrupted.
- `/effort` — Adjust reasoning depth. Dial it down for simple lookups, up for complex problems.
- `/clear` — Reset conversation context when it gets cluttered or you're switching tasks.
